## Getting Help

### GitHub Issues

For bugs, feature requests, or general questions:

1. Search [existing issues](https://github.com/kase1111-hash/learning-contracts/issues) to see if your question has been answered
2. If not found, [open a new issue](https://github.com/kase1111-hash/learning-contracts/issues/new/choose) using the appropriate template:
   - **Bug Report** - For reporting bugs or unexpected behavior
   - **Feature Request** - For suggesting new features or improvements

### GitHub Discussions

For broader discussions, ideas, or community support, use [GitHub Discussions](https://github.com/kase1111-hash/learning-contracts/discussions) (if enabled).

## Security Issues

**Do NOT report security vulnerabilities through public GitHub issues.**

Please see [SECURITY.md](SECURITY.md) for responsible disclosure guidelines.

## Response Times

This is an open-source project maintained by volunteers. While we aim to respond promptly:

- **Critical bugs/security issues**: Prioritized for quick response
- **General bugs**: Reviewed within 1-2 weeks
- **Feature requests**: Evaluated based on project roadmap
- **Questions**: Answered as time permits
